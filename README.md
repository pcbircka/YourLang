# YourLang

# Что такое YourLang?
YourLang — это приложение, которое изучает, как ты говоришь на родном языке, и автоматически создаёт персональный словарь и материал для изучения иностранного языка на основе твоей собственной речи.

# Цели проекта
- Упростить изучение языка, делая его естественным и основанным на собственной речи пользователя.
- Создавать персональный словарь, который реально нужен человеку.
- Помогать учить полезные слова и выражения, а не случайный учебный материал.
- Автоматизировать анализ речи и перевод основных слов на изучаемый язык.
- Сделать обучение более интересным и индивидуальным.

# Основная идея
Пользователь говорит на своём языке → приложение анализирует эту речь → автоматически создаёт персональный словарь и учебный материал на иностранном языке, основанный именно на том, как человек выражается.
